Weather-data-library

A library that communicates with www.openweathermap.org API

View on GitHub Download .zip Download .tar.gz

Project Goals

A library developed in C # which provide other developers with an easy and orderly access to weather data around the world. The library will include a code that communicates with the REStful Web Services site www.openweathermap.org.

Development Process

The development process was done by TDD methodology. It included the use of Git to manage both development and to share of the final product with the developers community by the presentation of the project for use on Www.github.org site. You can find video clips in English that explain how to work with the library on YouTube.
